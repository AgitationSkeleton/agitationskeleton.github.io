Welcome to REDchanit
This server features custom weapons. Check your loadout to see what's available. We're always adding more, too!
We also feature a number of custom player models.
As Soldier, type !grunt to become the TF2C Beta 4 Mercenary, !merc to become the Open Fortress Mercenary, or !driver to become the Driver.
As Demoman, type !janitor to become the TF2C Janitor.
As Civilian, type !scientist to become the TF2C Scientist.
SaySounds (woody and pingas) can be disabled with !sounds
Use ingame voting to change the level. Pause the game and click the vote icon on the menu to initiate a map vote.
