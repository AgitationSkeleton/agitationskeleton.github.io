Welcome to REDchanit: The Classic XPerience
This server is a vanilla TF2 experience to replicate the classic gameplay of release and console TF2.
Special classes, hauling, airblast, bunnyhopping and loadouts are disabled.
If you would like to play with the above features which are disabled, consider joining the other REDchanit TF2C server, which also offers a number of custom maps.
Use ingame voting to change the level.
Micspam allowed and encouraged!
